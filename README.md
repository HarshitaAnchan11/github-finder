# 🔍 GitHub Profile Finder

A responsive web app that lets you search any GitHub user and view their 
profile details and top repositories in real time.

🔗 **Live Demo**: https://harshitaanchan11.github.io/github-finder/
if doesn't work on a browser try a different browser. mine worked on microsoft edge but wasnt working on google chrome

## Features
- Search any GitHub username
- View avatar, bio, followers, following, and public repo count
- See top 6 repositories sorted by stars
- Responsive design — works on mobile and desktop
- Enter key support and loading state handling

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript (ES6+)
- GitHub REST API (no auth required)
- Deployed on GitHub Pages

## What I Learned
- Async/await and fetch API for real-time data
- DOM manipulation and dynamic rendering
- Error handling for failed API responses
- Responsive CSS layout with Flexbox
